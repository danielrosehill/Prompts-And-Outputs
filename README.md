#  Prompts & Outputs: A GPT Logbook

Some open-ended prompting experiments against GPT (at the time of writing, usually GPT 4o by OpenAI).

I capture these as a sort of loosely structured open-access journal, hoping that they contain some useful nuggets of information for future inquiry.

The file structure is `yy/mm`
